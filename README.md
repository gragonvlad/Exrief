# Exrief
Exrief is a simple grief plugin masked as AAC AntiCheat.

## Install
1. Download Exrief (https://github.com/CCBlueX/Exrief/releases)
2. Rename the jar to AAC.jar
3. Put the jar into the plugins directory
4. Start the server

## Commands

### Bukkit Commands
> /aac

### Exrief Commands

> *help

> *asconsole

> *damagespam

> *freeze

> *kill
